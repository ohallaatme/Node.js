Q: In your own words highlight the differences in functionality and purposes Client-Side and Server-Side code serves in a full-stack web application.

A: Differences include:

- Server side code runs in processes whereas client side runs in the browser

- Server side code doesn't render anything whereas client side does

- Server side code sends responses, client side makes requests

- Client side code handles interactions from the user, server side handles requests 

- Client side code uses script tags to split code, server side uses modules
